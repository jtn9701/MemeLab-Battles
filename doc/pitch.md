# Pitch
## Summary
Players will connect to a webpage and compete in a jackbox style game.
Players must create a meme, and everyone votes for the best one.

## Player
- 1 Host determines when to start the game
- Each player generates 1 meme to send to the server
- After server receives all memes, player votes and sends vote to server

## Server
- Handles player count/limit
- Handles vote count
- Handles API calls
- Sends data to Player and kiosk

## Kiosk
- Displays game lobby
- Displays votes for meme battle
- Displays Winner
